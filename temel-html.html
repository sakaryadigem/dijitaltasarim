HTML Temel Bilgileri Özeti

Web geliştirmeye başlamak için ihtiyacınız olan temel HTML bilgileri

HTML Nedir?

HTML (HyperText Markup Language), web sayfalarının yapısını oluşturmak için kullanılan standart işaretleme dilidir. Tarayıcılar HTML kodlarını yorumlayarak görsel web sayfalarına dönüştürür.

Temel HTML Yapısı

<!DOCTYPE html>
<html>
<head>
  <title>Sayfa Başlığı</title>
</head>
<body>
  <h1>Başlık</h1>
  <p>Paragraf</p>
</body>
</html>

Temel HTML Etiketleri

Başlık Etiketleri

<h1> - <h6>: Başlık etiketleri (h1 en büyük, h6 en küçük)

Metin Etiketleri

<p>: Paragraf

<strong> veya <b>: Kalın metin

<em> veya <i>: İtalik metin

<u>: Altı çizili metin

Bağlantı ve Görsel

<a>: Bağlantı (link)

<img>: Resim

Listeler ve Tablolar

Liste Etiketleri

<ul>: Sırasız liste

<ol>: Sıralı liste

<li>: Liste öğesi

Tablo Etiketleri

<table>: Tablo

<tr>: Tablo satırı

<td>: Tablo hücresi

<th>: Tablo başlık hücresi

<table>
  <tr>
    <th>İsim</th>
    <th>Yaş</th>
  </tr>
  <tr>
    <td>Ahmet</td>
    <td>25</td>
  </tr>
</table>

Formlar

HTML formları kullanıcıdan veri toplamak için kullanılır.

Temel Form Etiketleri

<form>: Form kapsayıcısı

<input>: Giriş alanı

<textarea>: Çok satırlı metin alanı

<select> ve <option>: Açılır liste

<button>: Buton

Input Türleri

type="text": Metin girişi

type="password": Şifre girişi

type="email": E-posta girişi

type="submit": Gönder butonu

Önemli Özellikler

id ve class

id: Bir öğeye benzersiz kimlik verir

class: Öğeleri gruplamak için kullanılır

Diğer Önemli Özellikler

href: Bağlantı adresi

src: Kaynak dosya yolu

alt: Alternatif metin (görseller için)

style: Satır içi CSS

<a href="https://example.com">Örnek Site</a>
<img src="resim.jpg" alt="Açıklama">
<p id="benzersiz" class="kirmizi">Metin</p>

Semantik HTML

Semantik HTML, içeriğin anlamını daha iyi tanımlamak için kullanılan etiketlerdir.

Semantik Etiketler

<header>: Sayfa veya bölüm başlığı

<nav>: Navigasyon menüsü

<main>: Ana içerik

<section>: İçerik bölümü

<article>: Bağımsız içerik

<aside>: Yan içerik

<footer>: Sayfa veya bölüm altbilgisi

Semantik etiketler, hem arama motorları hem de ekran okuyucular için daha anlamlı bir yapı oluşturur.
