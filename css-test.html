Temel CSS Bilgileri Özeti

Stil sayfaları ile web tasarımını öğrenmek için ihtiyacınız olan temel CSS bilgileri

📖 CSS Nedir?

CSS (Cascading Style Sheets), HTML elementlerinin görünümünü ve düzenini kontrol etmek için kullanılan stil dilidir.

CSS Söz Dizimi

seçici {
  özellik: değer;
}

CSS Ekleme Yöntemleri

Inline CSS: HTML elementi içinde style attribute'u ile

Internal CSS: <style> etiketleri arasında

External CSS: Harici .css dosyası ile

🎯 Seçiciler (Selectors)

Temel Seçiciler

Element: p, h1, div gibi HTML etiketleri

Class: .className şeklinde (birden fazla element)

ID: #idName şeklinde (tek bir element)

Gelişmiş Seçiciler

Gruplama: h1, p, div (virgül ile ayırma)

Çocuk: div > p (direkt çocuk element)

Torun: div p (tüm torun elementler)

/* Element seçici */
p { color: blue; }

/* Class seçici */
.kirmizi { color: red; }

/* ID seçici */
#baslik { font-size: 24px; }

🎨 Renk ve Tipografi

Renk Tanımlama

İsim: red, blue, green

HEX: #FF0000, #0000FF

RGB: rgb(255, 0, 0)

RGBA: rgba(255, 0, 0, 0.5) (alpha: şeffaflık)

Yazı Stilleri

font-family: Yazı tipi

font-size: Yazı boyutu

font-weight: Yazı kalınlığı

text-align: Metin hizalaması

h1 {
  color: #2c3e50;
  font-family: Arial, sans-serif;
  font-size: 2rem;
  text-align: center;
}

📐 Kutu Modeli (Box Model)

Her HTML elementi bir kutu olarak düşünülür:

İçerik (Content)

Kutu Model Özellikleri

width/height: Genişlik/Yükseklik

padding: İç boşluk

border: Kenarlık

margin: Dış boşluk

.kutu {
  width: 300px;
  padding: 20px;
  border: 2px solid #333;
  margin: 10px;
}

🎪 Flexbox ve Grid

Flexbox (Esnek Kutu)

Tek boyutlu düzenler için

1
2
3

CSS Grid

İki boyutlu düzenler için

A
B
C
D
/* Flexbox */
.container {
  display: flex;
  justify-content: space-between;
}

/* Grid */
.container {
  display: grid;
  grid-template-columns: 1fr 1fr;
}

✨ Animasyon ve Geçişler

Transition (Geçiş)

CSS özelliklerindeki değişikliklere yumuşak geçiş ekler

Animation (Animasyon)

Karmaşık animasyonlar için keyframes kullanır

Animasyon Örneği
/* Transition */
.button {
  transition: all 0.3s ease;
}

/* Animation */
@keyframes colorChange {
  from { background: #3498db; }
  to { background: #e74c3c; }
}
