HTML Örnekleri

Aşağıda temel HTML yapısının bir örneğini görebilirsiniz:

<!DOCTYPE html>
<html lang="tr">
<head>
  <meta charset="UTF-8">
  <title>Web Sayfam</title>
</head>
<body>
  <header>
    <h1>Web Sitem</h1>
    <nav>
      <ul>
        <li><a href="#">Ana Sayfa</a></li>
        <li><a href="#">Hakkımda</a></li>
        <li><a href="#">İletişim</a></li>
      </ul>
    </nav>
  </header>
  <main>
    <section>
      <h2>Hoş Geldiniz</h2>
      <p>Bu benim web sitem.</p>
    </section>
  </main>
  <footer>
    <p>Telif Hakkı © 2023</p>
  </footer>
</body>
</html>
